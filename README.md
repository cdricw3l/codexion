## Instructions

To compile the library and generate the static library.

```bash
make
```
or

```bash
make codexion
```
or

```bash
make all
```
To delete all object files
```bash
make clean
```

To delete all object files and the static library:
```bash
make fclean
```

To delete all object files, the static library and compile the library:
```bash
make re
```
